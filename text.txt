Задачи для лабораторной 2
Выполнить решение задачи на тестовой последовательности. Оформить вывод результатов (указать номер потока и номер строки).
Для анализа эффективности параллельных вычислений взять большой текстовый файл. Вычислить время решения задачи без использования потоков. Организовать решение задачи с использованием потоков: главный поток управления запускает N дочерних потоков, каждый из которых имеет номер k и ищет образец в строке k – 1, k – 1 + N, k – 1 + 2N и т.д. до конца файла. Задать количество потоков (N), произвести вычисления, рассчитать время. Определить размерность задачи приемлемую для организации параллельных вычислений. Перед началом параллельных вычислений файл можно прочитать в глобальную переменную.
1.	На каждой странице найти и вывести самую длинную строку.
2.	Объединить самые короткие строки (предполагается наличие ряда строк одной длины) в файл.
3.	Найти строки, где встречаются цифры и вывести их общее количество.
4.	Найти строки, где встречаются символы верхнего регистра и вывести их количество.
5.	Вычислить количество слов в строках, для каждого потока распечатать строки с большим значением.
6.	Заменить в каждом слове строк первую букву на заглавную.
7.	В каждой строке найти первый пробел, удалить все символы после него.
8.	Расставить строки каждой страницы текста в порядке возрастания их длины.
9.	Вычислить количество букв и цифр в строке, вывести строки с количеством цифр больше k.
10.	Сформировать слова из первых букв каждой строки.
11.	Вычислить количество гласных и согласных в строке.
12.	Найти и удалить пустые строки и лишние пробелы из строк.
13.	В тексте слова в каждой строке расставить в алфавитном порядке.
14.	В тексте строки на каждой странице расставить в алфавитном порядке.
15.	Найти в текстовом файле заданную строку. Какой поток её нашел?
16.	Расставить слова каждой строки в порядке возрастания их длины.
17.	Удалить самые короткие слова из каждой строки.
18.	Для каждых N – строк вставить каждую следующую строку в середину предыдущей.
19.	Самое длинное слово каждой строки поставить на первое место в строке.
20.	В каждой строке найти повторяющиеся слова, если таких нет подсчитать количество слов строки.
